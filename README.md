## 🔍 Overview
This project is an interactive **E-Commerce Sales Dashboard** built using Power BI to analyze sales performance, profit trends, and customer behavior.  
It helps businesses gain insights into key metrics and make data-driven decisions.

---

## 🎯 Objectives
- Analyze overall sales, profit, and quantity  
- Track monthly profit trends  
- Identify top-performing states and categories  
- Understand customer purchasing behavior  
- Analyze payment modes and sub-category performance  

---

## 🛠 Tools & Technologies
- **Power BI** – Data visualization  
- **Microsoft Excel** – Data source & preprocessing  

---

## 📈 Dashboard Highlights

- 💰 **Total Sales:** 438K  
- 📊 **Total Profit:** 37K  
- 📦 **Total Quantity Sold:** 5615  
- 📉 **Average Value:** 121K  

---

## 📊 Key Insights

- 🛍 **Clothing category** contributes the highest sales (63%)  
- 💳 **COD (44%)** is the most used payment method  
- 📍 **Maharashtra** leads in sales contribution  
- 📈 Profit shows fluctuations across months with peak performance in selected quarters  
- 🧾 Sub-categories like **Printers and Bookcases** generate higher profit
