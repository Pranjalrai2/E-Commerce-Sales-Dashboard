# 📊 E-Commerce Sales Dashboard

## 🔍 Overview
This project is a Power BI dashboard that analyzes e-commerce sales data to provide insights into revenue, profit, and customer trends.

## 🛠 Tools Used
- Power BI
- Excel

## 📈 Features
- Sales performance analysis
- Profit insights
- Category-wise breakdown

## 📁 File
Download the `.pbix` file and open it using Power BI Desktop.
